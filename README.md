# spring-boot-usuarios-crud
Ejemplo sencillo de webapp SSR con Spring Boot.

Este sencillo proyecto muestra cómo funciona Spring Boot para construir aplicaciones web SSR (server-side rendering), con vistas construidas con Thymeleaf.

Para ponerlo en marcha:
* Descarga el código o clona el repo
* Ábrelo con VS Code o tu IDE preferido
* Lanza el main() que encontrarás en SsrAplication.java. Spring Boot ya tiene un servidor web embebido.
* Lanza el servidor MariaDB con "docker-compose up" (tienes un docker-compose.yml en la raíz del proyecto)

Tu servidor debería arrancar con normalidad. La app estará en localhost:8080/usuarios
